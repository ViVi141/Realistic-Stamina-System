# 基准负重（Base Weight）分析

## 问题

当前模型中，"无负重"状态为0kg，这在现实中不合理。即使是"无负重"状态，角色仍然会携带基础物品（衣服、鞋子等）。

## 基准物品参考

### 基础物品（Base Items）

只包括衣服、鞋子等基础物品，不包括武器、装备等：

| 物品类型 | 重量（kg） |
|---------|-----------|
| 战斗服（BDU/ACU） | 0.8-1.0 |
| 战斗靴 | 0.5-0.6 |
| 其他（内衣、袜子等） | 0.1-0.2 |
| **总计** | **约1.36 kg** |

### 战斗负重（Combat Load）

战斗负重 = 基本战斗装备 + 额外装备（背包、补给等）

| 负重类型 | 重量（kg） |
|---------|-----------|
| 基本战斗装备 | 13.5-20.0 |
| 战术背包（含补给） | 10.0-15.0 |
| 额外弹药 | 2.0-5.0 |
| **总计** | **25.5-40.0 kg** |

### 最大负重（Maximum Load）

最大负重 = 战斗负重 + 额外装备

| 负重类型 | 重量（kg） |
|---------|-----------|
| 战斗负重 | 25.5-40.0 |
| 额外装备（帐篷、睡袋等） | 5.0-10.0 |
| **总计** | **30.5-50.0 kg** |

## 建议的基准值

### 方案1：基础物品（推荐）

- **基准负重（BASE_WEIGHT）**：**1.36 kg**
  - 包含：战斗服、靴子等基础物品
  - 这是"无额外负重"状态下的基础物品重量

- **战斗负重（COMBAT_ENCUMBRANCE_WEIGHT）**：**30.0 kg**
  - 包含：基准负重 + 武器、装备、战术背包、额外补给
  - 额外负重：28.64 kg

- **最大负重（MAX_ENCUMBRANCE_WEIGHT）**：**40.5 kg**
  - 包含：战斗负重 + 额外装备
  - 额外负重：39.14 kg

## 对模型的影响

### 当前模型（1.36kg基准，推荐）

- 基准负重（1.36kg）：1.5% 体重，速度下降0.3%，消耗增加2.3%
- 战斗负重（30kg）：33% 体重，速度下降6.6%，消耗增加50%
- 最大负重（40.5kg）：45% 体重，速度下降9%，消耗增加68%

**优势**：
- ✅ 更符合现实：角色总是穿着基础衣物
- ✅ 基准状态影响极小，几乎可以忽略
- ✅ 战斗负重和最大负重的计算保持不变
- ✅ 更符合"无额外负重"的概念（只有基础衣物）

### 方案2（15kg基准）

- 基准负重（15kg）：17% 体重，速度下降3.4%，消耗增加26%
- 战斗负重（30kg）：33% 体重，速度下降6.6%，消耗增加50%
- 最大负重（40.5kg）：45% 体重，速度下降9%，消耗增加68%

### 方案3（18kg基准）

- 基准负重（18kg）：20% 体重，速度下降4%，消耗增加30%
- 战斗负重（30kg）：33% 体重，速度下降6.6%，消耗增加50%
- 最大负重（40.5kg）：45% 体重，速度下降9%，消耗增加68%

## 推荐方案

**推荐使用方案1（1.36kg基准）**，原因：

1. **符合现实**：1.36kg是基础衣物（战斗服、靴子）的合理重量
2. **保持游戏平衡**：基准状态影响极小（速度下降0.3%，消耗增加2.3%），几乎可以忽略
3. **计算简单**：只需要在计算时减去基准重量
4. **符合概念**："无额外负重"指只有基础衣物，不包括武器、装备等

## 实现方式

### 方式1：在计算时减去基准重量（推荐）

```enforce
float currentWeight = characterInventory.GetTotalWeight();
float effectiveWeight = Math.Max(currentWeight - BASE_WEIGHT, 0.0);
float bodyMassPercent = effectiveWeight / CHARACTER_WEIGHT;
```

**优势**：
- ✅ 不需要修改游戏引擎的负重系统
- ✅ 保持战斗负重和最大负重的定义不变
- ✅ 计算简单清晰

### 方式2：直接使用总重量

```enforce
float currentWeight = characterInventory.GetTotalWeight();
float bodyMassPercent = currentWeight / CHARACTER_WEIGHT;
```

**优势**：
- ✅ 更简单
- ✅ 但需要调整战斗负重和最大负重的定义

## 结论

**推荐使用1.36kg作为基准负重**，在计算时减去基准重量，这样：
- 基准状态（1.36kg）：有效负重0kg，几乎无影响（速度下降0.3%，消耗增加2.3%）
- 战斗负重（30kg）：有效负重28.64kg（32%体重），速度下降6.4%，消耗增加48%
- 最大负重（40.5kg）：有效负重39.14kg（43%体重），速度下降8.6%，消耗增加65%

这样既符合现实（只有基础衣物），又保持了游戏平衡。
